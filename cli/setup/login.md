# Login

Before you can use `sym`, you'll need to let us know who you are! 

```bash
$ sym login
Org: your-org-slug-here
Email: you@your-org.com
```

{% hint style="info" %}
If you don't know your organization's slug, please contact support.
{% endhint %}
